Problem

At the enterprise, production records, hourly work logs and quality control data were mostly kept on paper. All this information had to be manually transferred to the accounting program, making payroll calculations, production statistics tracking and planning complicated and time-consuming. Additionally, employee registration for company events was handled through separate channels, with data processed manually as well.

Solution

We created a web-based application for farm production accounting, statistics tracking, reporting and event registration. The application was integrated with scales and label printers, featured a role-based access system for both management and employees. Salary-related data can be exported directly to the main accounting program.

Result

The application significantly accelerated and simplified enterprise accounting, increasing productivity through timely data processing and analysis. Manual data entry was eliminated and real-time insights became available for better decision-making.

Technical Part

Oracle Database and APEX deployed in Docker container, with Nginx as a local proxy server and Cloudflare handling external traffic. The application was hosted on cloud infrastructure with a custom domain and utilized a dedicated SMTP server for notifications. A separate backup server ensured data reliability. For security reasons, some technical details cannot be disclosed.

BeeBooking

Problem

The company recruits seasonal workers from Finland and other countries, but the application process was inefficient and fragmented. Employees applied via email without a standardized application form or processing system. Each application required an individual response, and employee data had to be manually transferred to the accounting system. When the employer organized transportation, passenger lists were also created manually, adding to the administrative burden.

Solution

We created a web application for submitting job applications, processing them, and managing recruitment workflows. The system includes an integrated ticket booking feature for employee transportation. Applicants can attach scans of required documents directly to their application forms. They automatically receive a notification about the change in the status of the application. Employee's data can be exported to the accounting system automatically.

Result

The application streamlined the entire recruitment process, handling over 500 job applications annually. Manual data entry and email-based communication were replaced with an efficient, centralized system that saves time and reduces errors.

Styres

Problem

Small businesses needing warehouse management systems face a challenging market situation. Enterprise-level ERP solutions with WMS modules cost 100+ euros monthly and offer far more complexity than needed. Simpler alternatives priced at 18-24 euros per month have significantly limited features or hidden event-based charges. Many small businesses (providing storage services like tire warehouses, service stations, or frozen berries storages) still rely on outdated methods such as spreadsheets due to the lack of affordable, targeted solutions.

Solution

We developed Styres, a modern SaaS warehouse management application designed specifically for small businesses. The platform provides essential modules including warehouse management, goods tracking, inventory control, customer management, and data import/export capabilities. The system features a simple, intuitive interface that delivers targeted information without unnecessary complexity. Optional modules for industry-specific features (such as tire storage services) allow for customization based on business needs.

Result

Styres delivers a targeted solution for small businesses seeking professional warehouse management without the burden of expensive, overcomplicated systems.

Technical Part

The architecture of the application supports horizontal scaling through multiple container instances and database schema separation for multi-tenant deployment. Cloud-hosted infrastructure.
